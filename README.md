##Feels Like
A simple Weather App for Android. This app was made as part of Udacity Android Nanodegree.

##Installation
This app uses an API key from http://openweathermap.org/api.

##Features:
- Material design
- Adaptive and responsive UI
- Offline usage (with SQLite)
